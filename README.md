# MarketplaceApi
Репозитория API маркетплейсов на языке c#
### Концепция
Для каждого маркетплейса, будет создано отдельное решение.
Базовый принцип DI . Библиотека должна предоставлять публичные интерфейсы для взаимодействия с маркетплейсами
